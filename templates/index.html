About our project

Deepfake technology refers to synthesized or superimposed images, videos created by artificial intelligence using existing images and videos. Using machine learning, this technology could theoretically create convincing footage of a person performing an act that never actually occured in real life. Since the appearance in late 2017, many open-source deepfake generation methods have emerged, leading to a growing number of synthesized media clips. While many arelikely intended to be humorous, others could be harmful to individuals and society. This project aims to detect deepfakes using deep convolutional neural networks with the help of machine learning frameworks such as PyTorch, Tensorflow etc.
